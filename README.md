# Wild Ember Digital starter site

Open the folder in VS Code and launch `index.html` with Live Server.

## Included
- Home
- Portfolio
- Services
- About
- Contact with Netlify Forms markup
- Privacy starter
- Terms starter
- Responsive CSS
- Mobile navigation

## Replace before launch
- `hello@wildemberdigital.ie`
- Placeholder project links (`href="#"`)
- Portfolio copy and images
- Photo placeholder
- Privacy and terms wording so it matches your actual business setup

## Netlify form
The contact form already includes `data-netlify="true"`. Deploy the whole folder to Netlify, submit one test enquiry, and check the Forms area in the Netlify dashboard.
